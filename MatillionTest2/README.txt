To run this application, use a Java IDE (such as IntelliJ/Eclipse) or open Command Prompt, change to the artifacts directory and type "java Main x x x " x being the parameter of your choice.

This application requires three parameters for operation which are split using spaces.
The first parameter needs to be a numerical value for department ID"
The second parameter needs to be the pay type
The final parameter needs to be the education level
For example, type 'java Main 18 Hourly Graduate'
