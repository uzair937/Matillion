Matillion Java Test 1 by Uzair Foolat 20/04/2019

To run this application, use a Java IDE (such as IntelliJ/Eclipse) or open Command Prompt, change to the artifacts directory and type "java Main x y" x being the first String and y being the second String.

This application compares two Strings provided, with spaces to separate them and returns a single numerical value with the differences between each String.
